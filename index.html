✂️ CommandMove

Cut & Paste
files on Mac.

macOS only lets you copy files. CommandMove adds the missing Cmd+X so you can actually move them.

Download v1.0 GitHub →
Cut files
⌘ X
→
Move them
⌘ V
✂️

Works with your muscle memory

Uses the same Cmd+X / Cmd+V shortcuts you already know. No new keybindings to learn.

📍

Lives in your menu bar

Click the scissors icon to cut, paste, clear clipboard, or quit. Stays out of your way.

🛡️

Handles conflicts

Duplicate filename at the destination? It appends a number automatically. Skips files already there.

⚡

Lightweight Swift app

No dependencies. Just Swift, CGEvent taps, and AppleScript. Compiles with a single command.

Get started

Download DMG
1
Download and open the .dmg file, then drag CommandMove into Applications
2
Open System Settings → Privacy & Security → Accessibility and toggle CommandMove on
3
Allow Automation access when prompted so it can control Finder
4
Select files in Finder, press ⌘X, navigate somewhere, press ⌘V — done!
